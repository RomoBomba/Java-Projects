### 7. URL Shortener
- Описание: Веб-приложение для сокращения длинных URL-адресов. Пользователи могут регистрироваться для получения API-ключа, создавать короткие ссылки, просматривать статистику переходов. Администраторы могут удалять любые ссылки. Cервис перенаправляет пользователей на оригинальные URL при переходе по короткому ключу.
- Технологии: Java 23, Maven, Spring Boot, Spring Security, Spring Data JPA, PostgreSQL, SpringDoc OpenAPI 2.5.0 (Swagger UI), Lombok
- Что я изучил: Реализация REST API с использованием Spring MVC, аутентификация через API-ключи с Spring Security, работа с базами данных через Spring Data JPA, обработка ошибок с помощью @ControllerAdvice, применение DTO для запросов и ответов

#### Ключевые особенности:
- Аутентификация по API-ключу в заголовке X-API-KEY
- Ролевая модель (USER, ADMIN)
- Генерация уникальных ключей для коротких ссылок
- Подсчет кликов и отслеживание времени последнего перехода
- Интеграция Swagger UI с поддержкой аутентификации
